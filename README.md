# Multi-Campus University Network Infrastructure
**Course:** CSE405 - Computer Networks  
**Institution:** East West University, Dhaka, Bangladesh  
**Instructor:** Dr. Anisur Rahman  
**Semester:** Spring 2026

## 📌 Project Overview
This project involves the design and implementation of an 8-campus network for "Apex University." It features a centralized server room, OSPF dynamic routing, and automated IP addressing.

## 🛠️ Technical Specifications
* **Software:** Cisco Packet Tracer v9.0.0.0810
* **Routing Protocol:** OSPF (Open Shortest Path First)
* **Services:** DHCP Relay, DNS, HTTP (Web Server)
* **Addressing:** Mixed Class A, B, and C subnets
* **Connectivity:** Hybrid Wired/Wireless (WPA2-PSK)

## 🌐 Network Features
* **Centralized DHCP:** One server provides IPs for all 8 campuses.
* **University Website:** Accessible via `www.apex.edu.bd`.
* **Campus 8 Integration:** Successfully configured with automatic IP/Subnet via relay.
